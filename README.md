# 📚 マイライブラリ（蔵書管理アプリ）

## 概要
Python と Streamlit、SQLite を用いて作成した蔵書管理アプリです。  
本の追加・編集・削除に加え、読書ステータスや評価（★）を管理できます。

---

## 主な機能
- 📖 蔵書一覧表示（カードUI）
- 🔍 タイトル・著者による検索
- ✏️ 蔵書情報の編集・削除
- ➕ 新しい本の追加
- ⭐ 評価（1〜5）
- 🔁 再読フラグ管理
- 👤 ユーザー名入力による簡易ログイン

---

## 使用技術
- Python
- Streamlit
- SQLite3
- Pandas
